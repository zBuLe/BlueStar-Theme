# BlueStar-Theme
a simple theme

modify the _config.yaml file

install the theme

'''
remote_theme: zbule/bluestar-theme
'''

theme settings

```
font_base: 16px
panel_width: 22em
accent_color: blue
content_max: 68em
```

Site Navigation

```
# Dynamic sidebar navigation
nav:
  - title: 🧪 Link1
    url: /
  - title: 📜 Link 2
    url: /link2
  - title: 🛠️ about
    url: /about
  - title: 🗺️ Roadmap
    url: /roadmap
```

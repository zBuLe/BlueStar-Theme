# BlueStar-Theme
a simple theme

modify the _config.yaml file

install the theme
```
remote_theme: zbule/bluestar-theme

plugins:
  - jekyll-remote-theme
```

theme settings

```
font_base: 16px
panel_width: 22em
accent_color: blue
content_max: 68em
```
